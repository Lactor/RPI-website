Tree of Knowledge

The tree of knowledge is a project of mine to create a place where you can learn anything, whether Physics, Chemistry, Mathematics or any other subject.

I want to make it a plataform were everyone can add and correct what is already available. Like a Wikipedia however with the goal of being a place you can come to learn rather to consult. While Wikipedia is a great source to consult information, check facts, it is a very hard place to learn a new topic.

For this reason I decided to create this project, with the idea of using it to teach others what I know, and learn from others.
